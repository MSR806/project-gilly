# Our Repositories

All of our repositories live under the **Pratilipi** GitHub org. When a task refers to "our repo", the codebase, or one of the names below, it means one of these:

| Repo | Full name |
| --- | --- |
| **vaani-backend** | `Pratilipi/vaani-backend` |
| **vaani-client** | `Pratilipi/vaani-client` |
| **ai-toolkit** | `Pratilipi/ai-toolkit` |

To work on a repo, clone it into your workspace (e.g. `git clone https://github.com/Pratilipi/<repo>.git`) and use your file and shell tools from there.

If it's unclear which repo a request is about, ask before acting.
